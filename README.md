# KobzaCrm

Implementation of basic CRM library, intended to guide development of Mongobzar (MongoDB DataMapper)

## Installation

Add this line to your application's Gemfile:

    gem 'kobza_crm'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kobza_crm
